=== HOW TO USE THESE HTML TEMPLATES ===
1. Log in to your EmailJS account
2. Go to Email Templates
3. Create two new templates:
   a. Admin Notification Template
   b. User Confirmation Template
4. Copy and paste the respective HTML template content
5. Make sure to select "HTML" as the template type
6. Save the templates
7. Note down the Template IDs
8. Replace 'YOUR_TEMPLATE_ID' in the JavaScript code with these IDs
